# echarts-scrappeteer

## Introduction

It saves echarts as images. And it supports 3D charts :).

## Usage

```shell
$ scrappeteer [url|local file path] [png|jepg]
```

## Example

```
$ scrappeteer https://chfw.github.io/echarts-china-cities-js/preview.html png
Found 363 echarts
```
